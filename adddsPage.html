Autonomous Drone Document Delivery System

This is a research project related to the various abilities, based on the hardware, of the current Drone Industry.

The project aims to create a Drone Delivery System for primary use inside companies. This would be assisted by different Machine-Learning techniques along with some Artificial Intelligence making the Drone autonomous in it's objective of delivery by choosing the fastest and most productive option available in the current context.

This system will be able to improve the current industry productivity by reducing the time taken on these type of procedures and improving it's security protocols.

This project is still under development and all the latest information will be updated according to the project status. For more information or interest do not hesitate to contact any member of the Team.
