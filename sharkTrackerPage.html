Shark Tracker

This is my first year project. Created by the team "Talent", including me and the other three teamates.

My Role

Technical Lead

To design and work on the software functionalities, such as the Login in system, user security, getting data from APIs and some other backend features.

Group Manager

To organise the best works to teammates and progress control.

Description

We have to create a software to track the location of sharks with the same layout as the requirement. Base on that, we added even more functions to the application. Such as...

  • Real time loading page and progress bar
  • Login system, hashed and salted optain better security.
  • Different users(researchers) can follow any sharks and store into their own account for later uses.
  • Implemented Google Maps API to display locations of all your followed sharks.
  • "Shark Of The Day" randomly pick a shark from the server and display with a real time count down.
  • We implemented a web browser to show the video about the "Shark Of The Day" shark if there is a video link in the server.
